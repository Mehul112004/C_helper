# Phase 5: LLM Confirmation Pipeline & Confirmed Signals

## Goal
The LLM confirmation pipeline serves as the final algorithmic filter for generated trade candidate signals. Candidate signals generated by strategy scripts flow through a local LLM via LM Studio. The LLM parses recent OHLCV, indicators, S/R zones, and outputs a structured JSON response to either CONFIRM, REJECT, or MODIFY the signal.

## Implementation Details

### LLM Specifications
- **Model:** `qwen/qwen3.5-9b` (Qwen 2.5/3.5 9B variant).
- **Endpoint:** Local LM Studio server `http://localhost:1234/v1/chat/completions`.
- **Strengths:** Excellent logical and mathematical reasoning to process indicator data, S/R bounds, and precise JSON schema adherence.

### Backend Architecture
1. **`app/core/llm_client.py`**:
   - `LLMClient` class wrapping the LM studio API endpoint.
   - **Context Builder**: Formats the last 30 current-timeframe candles, last 10 higher-timeframe candles, current indicators, S/R zone proximity, and the raw SetupSignal block.
   - **JSON Schema Engine**: System prompt forces strict Pydantic JSON validation ensuring predictable `verdict` and `reasoning`.

2. **`app/core/llm_queue.py` (Background Queue)**:
   - Uses a self-contained Python `asyncio` background queue.
   - Designed to avoid blocking the main live Binance WebSocket scanner thread.
   - **Retry Fallback**: If LM Studio times out or is offline, signals are backed off and retried every 60 seconds, retaining a `PENDING` state.

3. **`app/models/db.py` (Database Persistence)**:
   - Contains a `ConfirmedSignal` table to persist outputs.
   - Attributes include `verdict_status` (CONFIRMED, REJECTED, MODIFIED), `reasoning_text`, and updated price levels.
   - **Immediate Modify Upgrade:** If the verdict is `MODIFY`, the system automatically adjusts the signal parameters and treats it as a validated signal.

4. **Frontend & API Hooks**:
   - `GET /api/system/lm-studio-status`: Endpoint to ping local LLM health.
   - Expansive reasoning badge in UI to display why a signal was confirmed.
